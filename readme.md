## Comandos Artisan

Crear un nuevo proyecto Laravel 5.2
```javascript
composer create-project --prefer-dist laravel/laravel nombreCarpetaDestino "5.2.*"
```
Ejecutar el Servidor Artisan:
```javascript
php artisan serve
```
